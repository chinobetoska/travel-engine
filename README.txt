Trip Fixed Starting Dates Countdown is an extension for WP Travel Engine plugin to add a fixed starting dates countdown to desired trips. It is dependent on the Trip Fixed Starting Dates add-on.

== Description ==

This extension allows you to add a fixed starting dates countdown to desired trips. It is dependent on the Trip Fixed Starting Dates add-on.

Trip Fixed Starting Dates Countdown adds a Trip Starting Countdown widget in the widgets section and a Trip Countdown Configuration meta box in the trip dashboard. It is dependent on Trip Fixed Starting Dates addon causing it to work only on the trips that have Fixed Starting Dates configured.
